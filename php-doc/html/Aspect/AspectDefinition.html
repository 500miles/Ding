AspectDefinition

Description
Description | Vars (details) | Methods (details) Constants (details)

This class is used when reading the bean definition. Aspects will be constructed and applyed using this information, you may thing of this as some kind of Aspect DTO created somewhere else and used by the container to assemble the final bean.

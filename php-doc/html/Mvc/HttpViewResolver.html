HttpViewResolver

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • IViewResolver
  • ILoggerAware

Http view resolver. It will try to map the given action name to a full path in the filesystem, accoding to the path, prefix, and suffix configured.

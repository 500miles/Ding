MessageSourceImpl

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • IMessageSource
  • IResourceLoaderAware

Default implementation for a MessageSource. Will try to find a file called bundle_locale.properties and resolve message id's as standard key values in a properties file.

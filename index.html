Ding by marcelog

DI ( Dependency Injection ), AOP ( Aspect Oriented Programming ), lightweight, simple, and quick MVC ( Model View Controller ), syslog, tcp client and server, with non blocking sockets, timers, and custom error, signal, and exception handling, PAGI Integration (for asterisk gateway interface or agi), PAMI Integration (for asterisk management). Needs PHP 5.3, very similar to seasar, spring ( java ) . Tested with doctrine2! ;)

Requirements

  • PHP 5.3.3+
  • Extensions: posix, pcntl, simplexml, sockets, pcre, reflection
  • log4php is an optional requirement, since Ding only logs for debug purposes. It will be used only if its found in the include_path.

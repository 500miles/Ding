The PamiHelper

This helper is designed to be a bean of scope singleton that uses the PAMI framework to let a listener of your choice be called with any asterisk manager events.

Class

Ding\Helpers\PAMI\PamiHelper

Description

This bean has the properties "handler", "host", "port", "username", and "password". Handler is a reference to another bean that must implement Ding\Helpers\PAMI\IPamiEventHandler.

The TimerHelper

This helper is designed to be a bean of scope prototype, and act as a timer, calling your callback every some milliseconds.

Class

Ding\Helpers\Timer\TimerHelper

Description

This bean has the properties "handler", and "milliseconds". Handler is a reference to another bean that must implement Ding\Helpers\Timers\ITimerHandler. The timer can be start()'ed and stop()'ed as needed (see the php-doc api for more details).
